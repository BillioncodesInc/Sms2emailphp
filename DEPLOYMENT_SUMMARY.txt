1. COMMIT TO GITHUB
   ```bash
   git add .
   git commit -m "feat: add Render.com deployment configuration"
   git push origin main
   ```

2. DEPLOY TO RENDER
   a. Go to: https://dashboard.render.com
   b. Click "New +" → "Blueprint"
   c. Connect your GitHub repository
   d. Render will auto-detect render.yaml
   e. Click "Apply" to create both services

3. CONFIGURE ENVIRONMENT VARIABLES
   Set these in Render Dashboard for backend service:
   - SMTP_SERVICE=Gmail
   - SMTP_USER=your-email@gmail.com
   - SMTP_PASS=your-app-password
   - ALLOWED_ORIGINS=https://your-frontend.onrender.com

4. ACCESS YOUR APP
   - Frontend: https://se-gateway-frontend.onrender.com
   - Backend: https://se-gateway-backend.onrender.com
